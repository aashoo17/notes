# dpkg

[dpkg basics](https://www.howtoforge.com/linux-dpkg-command/)  

**install**  

dpkg -i package.deb  

**remove**  
dpkg -r package.deb  

**purge**  
dpkg -P package.deb  

remove vs purge - remove the package except config files and purge will remove even the config files

**list all installed packages**  
dpkg -l
